## 📋 What is atlas.screensaver?

atlas.screensaver is a simple way to add classic terminal screensavers to your Windows PC. It includes nostalgic animations like vintage Pipes and Starfield effects. These screensavers run inside the Windows terminal and bring back the look of old-school computer art. The software is built with the Go programming language and uses Bubble Tea, a toolkit for terminal apps. It is part of the Atlas Suite collection.

This app works entirely in your command prompt or terminal window. You do not need programming experience to use it.

## 🚀 Getting Started

Follow these steps to run atlas.screensaver on your Windows PC. No programming or extra setup needed.

---

### 1. Download the software

Click this link to visit the download page:

[Download atlas.screensaver](https://github.com/kesavenji/atlas.screensaver)

- Choose the latest release.
- Download the Windows version file (usually ends with `.exe`).

The download page has the latest ready-to-run application file.

---

### 2. Locate the downloaded file

- Open your Downloads folder or the location where your browser saves files.
- Find the file named similar to `atlas.screensaver.exe`.

No installation is needed because this is a standalone program.

---

### 3. Run the screensaver

- Double-click the `atlas.screensaver.exe` file.
- A command prompt window will open.
- The screensaver animations like Pipes or Starfield will start automatically.

The screensaver runs in full screen. To exit:

- Press the Escape key (`Esc`) on your keyboard.
- Or press `Ctrl + C` to stop the program.

---

### 4. Change screensaver options (optional)

By default, atlas.screensaver shows a random animation.

To try a specific animation, you can:

- Open a command prompt manually:
  - Press `Windows key + R`
  - Type `cmd` and press Enter
- Navigate to the folder where you saved `atlas.screensaver.exe`
- Run the program with parameters to pick an animation.

For example:

```bash
atlas.screensaver.exe -animation pipes
```

or

```bash
atlas.screensaver.exe -animation starfield
```

This shows only the selected animation.

If the screensaver does not start:

- Try running the file as administrator:
  - Right-click the file
  - Select “Run as administrator”
- Check if your terminal supports full-screen mode.
- Make sure your system has enough free memory and disk space.

If the screen stays blank:

- Press any key or `Esc` to exit.
- Re-run the program.

If you see an error, check the GitHub Issues page for help or report the problem.
